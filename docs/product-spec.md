# Genzite – AI-Powered No-Code Business Application Builder & Dynamic CMS

## Product Objective

Genzite is a next-generation AI No-Code platform that allows non-technical users to create and operate fully functional web applications using natural language.

Instead of just generating static website interfaces, Genzite can automatically generate:
- Frontend user interfaces
- Backend APIs
- Dynamic CMS
- Dynamic data structures
- Business workflows
- Content management systems
- AI-powered business features

Users simply describe their requirements in natural language, for example:
- "Create an online flower shop website"
- "Create a clinic management system"
- "Create a personal portfolio website"
- "Create an AI CV Builder and Recruitment Portal"

The system utilizes Google Gemini to analyze requirements and automatically generate the appropriate application structure, data schema, and UI.

---

## Core Business Goals

### 1. AI Website & Application Generation
Enables users to create various types of applications without writing source code:
- E-commerce Website
- Restaurant Website
- Portfolio Website
- Booking System
- CRM Mini System
- Recruitment Portal
- AI CV Builder
- Learning Management System (LMS)
- Landing Page
- Internal Business Dashboard

### 2. Dynamic CMS with JSONB Architecture
The system supports dynamic data modeling through PostgreSQL JSONB. Users can create and modify:
- Forms
- Content Types
- Collections
- Business Objects

on the fly during runtime without requiring database migrations.

### 3. AI Resume & Recruitment Intelligence

#### AI CV Builder
- Create CVs from experience descriptions
- Generate CV content using AI
- Publish CVs online

#### AI CV Analysis
- Upload CV + Job Description
- Analyze compatibility match

#### AI Skill Matching
- ATS Score calculation
- Missing Skills Detection
- Keyword Optimization

#### AI Interview Generator
- Technical Questions
- Behavioral Questions
- Position-specific Questions (based on CV + JD)

#### AI Mock Interview
- AI acts as the interviewer
- Scores candidate responses
- Provides improvement feedback

#### AI Career Coaching
- Recommends learning paths and career development roadmaps
- Based on CV, JD, interview results, and career goals

---

## Target User Groups
- **Non-technical users / Creators**: Build complete web applications using only natural language.
- **HR / Recruiters**: Utilize the AI Recruitment Platform for CV analysis and interviewing.
- **Developers**: Extend APIs, create custom widgets, register new layouts.

## Design Philosophy
UI/UX aesthetic MUST be **cozy, user-friendly, and home-oriented**. Strictly reject harsh, technical, or traditional IT-dashboard designs. The platform should feel like a welcoming community.

---

## Expected Outcomes
Upon completion, Genzite will become an AI No-Code platform that allows users to create and operate real-world web applications and advanced AI features using only natural language. This will significantly reduce development time from weeks to minutes, while supporting various business domains on a single unified platform.
